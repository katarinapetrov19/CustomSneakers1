# Custom Sneakers Portfolio

Portfolio website for Sebastian Wagner - Custom Sneaker Artist & Mural Designer

## Features

- Responsive design for desktop, laptop, and mobile devices
- Interactive gallery with hover effects
- Scroll-based dark theme transition in About section
- Parallax animation on hero section
- Project detail pages

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure

```
CustomSneakers/
├── index.html          # Homepage
├── project.html        # Project detail pages
├── project2.html
├── project3.html
├── project4.html
├── style.css          # Main stylesheet
├── img/               # Image assets

```
